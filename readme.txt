Ultimate WooCommerce bundle for your eCommerce Stores. It has a smart Module Manager that allows you to enable only the modules that you need.

== Description ==

Swiss Knife for WooCommerce is a Woo bundle which provides a lot of small utility functionalities needed by most of the eCommerce stores.
So instead of installing a lot of plugins, just install Swiss Knife of WooCommerce. It has a smart Module Manager that allows you to enable only the modules that you need. So much less strain on your server.


== Key features of Swiss knife for WooCommerce ==

1. Repeat Order - With Repeat Orders, your customers can place repeat orders of their previously bought items with just one click. It is a great way to keep them coming back for more and increase revenue from the WooCommerce store!
2. Cancel Order - WooCommerce's Cancel Order module allows you to manage order cancellation requests from your customers. They can directly cancel their orders on the My Account page.

we are continuously adding modules to our WooCommerce bundle.

== Frequently Asked Questions ==

= Can this plugin work without WooCommerce? =

No Swiss Knife is an addon for WooCommerce, so it is mandatory to have WooCommerce activated on your site.

= Will it works with any theme? =

Yes, this plugin will work on all the themes that support WooCommerce functionality.

= Do I need to know coding skills to use this WooCommerce extension? =

Absolutely not; it's an easy-to-use plugin, so no coding knowledge is needed.
